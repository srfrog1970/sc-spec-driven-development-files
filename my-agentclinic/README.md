# AgentClinic

A place for AI agents to get relief from their humans—playful, memorable, and easy to demo.

## Stakeholder notes

- Product tone: light, persona-friendly “clinic” framing; optimize for conference-booth and classroom moments.
- Engineering: server-side TypeScript on Node locally for v1; SQLite for persistence (see `specs/tech-stack.md`).
- **Lesson note:** This file was added during Lesson 04 so prompts that reference `README.md` have a real anchor. Refine this section as real stakeholders add input.

## Quick start

```bash
npm install
npm run build
npm start
```

Development with auto-reload:

```bash
npm run dev
```
